Titanic Data Analysis 🚢

1. Objective

Analyze survival patterns of Titanic passengers using data storytelling.

2. Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

3. Key Insights
- Women survived more than men
- Higher class passengers had better survival
- Fare influenced survival chances

4. 📁 Dataset
Titanic dataset from Kaggle
